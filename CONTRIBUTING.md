# Contributing to Awesome NVIDIA NIM

Thanks for taking the time to contribute!

## How to add a resource

1. Fork this repository.
2. Edit `README.md` — add your link to the most relevant section.
3. Use this format:
   ```
   - [Name](https://link.com) — One sentence description. No trailing period needed.
   ```
4. Open a pull request with a short description of what you added and why it belongs.

## Quality guidelines

**Include:**
- Publicly accessible resources (no paywalls, no sign-up gates for the core content)
- Actively maintained projects (last commit or update within 12 months)
- Resources directly relevant to NVIDIA NIM — inference, deployment, integration, or tooling

**Do not include:**
- Broken links
- Duplicate entries (search before adding)
- Pure self-promotion with no working code or content
- Resources that only tangentially mention NIM

## Sections

If your resource doesn't fit an existing section cleanly, propose a new section in your PR description. New sections need at least 3 items to be worth creating.

## Code of conduct

Be respectful. Maintainers may edit your submission for formatting consistency without explanation.
